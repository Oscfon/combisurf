<h1><p align="center">combisurf</p></h1>

combisurf is a Python package for working with cell decompositions of surfaces
in [SageMath](https://sagemath.org).

combisurf is free software, released under the [GPL v2 (or later)](./COPYING).

We welcome any help to improve combisurf. If you would like to help, have
ideas for improvements, or if you need any assistance in using combisurf,
please don't hesitate to [contact us](https://flatsurf.github.io#contact).
